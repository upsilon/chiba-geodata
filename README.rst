=============
chiba-geodata
=============

これは何？
==========

千葉市がオープンデータとして公開してゐる情報のうち、緯度経度などの地理的な情報を持つデータを `GeoJSON <http://geojson.org>`_ 形式に変換して再配布する **非公式な** プロジェクトです。

ライセンス
==========

再配布する GeoJSON データは `CC BY 2.1 JP <http://creativecommons.org/licenses/by/2.1/jp/>`_ ライセンス下で利用できます。
